## Notes & Caveats

### Test Before Adding
- **Manual testing recommended** for all feeds before database insertion
- Verify feed format (RSS 2.0 vs Atom)
- Check for paywalls or authentication requirements
- Confirm content is excerpt-only compliant

### Feeds That Don't Exist (Researched but unavailable)
- **Reuters Politics RSS** - Discontinued (only wire service feeds available)
- **Axios Politics RSS** - No native RSS (requires third-party generator)
- **HuffPost Politics RSS** - Discontinued
- **Slate Politics RSS** - Discontinued

### Google News RSS Format
- Uses `/rss/search?q=QUERY&hl=en-US&gl=US&ceid=US:en` format
- Can customize queries for specific topics/people
- Aggregates from multiple sources
- May contain duplicate stories (rely on clustering)

### Compliance Considerations
- All new feeds should use **5000 char limit** (matching existing standard)
- All should be **excerpt-only** (`allow_full_text = false`)
- Test that RSS content respects fair use limits
